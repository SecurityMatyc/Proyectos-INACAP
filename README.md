Este repositorio contiene una recopilación de distintos proyectos, actividades y trabajos desarrollados durante el proceso de formación académica y práctica. 
Su propósito es reunir en un solo espacio los avances realizados en diversas áreas del desarrollo y la tecnología, reflejando el progreso y la aplicación de conocimientos a lo largo del tiempo de estudio.

Cada carpeta dentro del repositorio corresponde a un trabajo independiente, lo que permite revisar y comprender de manera ordenada los resultados y enfoques de cada experiencia de aprendizaje.
